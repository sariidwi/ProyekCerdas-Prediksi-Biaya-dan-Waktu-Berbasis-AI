# ProyekCerdas-Prediksi-Biaya-dan-Waktu-Berbasis-AI

**Project Overview:**
Proyek ini bertujuan untuk memitigasi risiko keterlambatan pada industri konstruksi dengan memanfaatkan Machine Learning. Sistem ini memprediksi kemungkinan keterlambatan proyek berdasarkan variabel-variabel historis, sehingga membantu manajer proyek dalam pengambilan keputusan yang lebih akurat.

**Role: Team Leader & System Analyst**

**Key Responsibilities:**

1. Leadership: Memimpin tim dalam siklus pengembangan model AI, mulai dari data gathering hingga evaluasi model menggunakan metode SCRUM.
2. System Analysis: Menganalisis parameter teknis yang mempengaruhi keterlambatan proyek dan menyusun alur logika data (Data Flow).
3. Technical Documentation: Menyusun laporan teknis komprehensif yang merinci arsitektur model, hasil pengujian, dan rekomendasi implementasi.
4. Collaboration: Menjembatani komunikasi antara tim pengembang model dan stakeholder untuk memastikan output AI relevan dengan kebutuhan industri konstruksi.

**Tech Stack:** Python, Scikit-Learn (Random Forest), Jupyter Notebook, Google Colab.
